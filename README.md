### 開発
#### dotnetコマンド実行(powershell)
``` powershell
# Developmentモードで実行
$Env:ASPNETCORE_ENVIRONMENT = "Development"
dotnet run
```

#### grunt task
``` powershell
# lessのコンパイル
grunt less

# ファイルの変更を監視
grunt watch
```
